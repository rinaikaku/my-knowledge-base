# 全局工作指引

## 文档语言

- 所有新建或更新的文档、知识库条目、分析报告、说明材料与交付物默认使用中文。
- 仅当用户明确指定其他语言，或源材料中的原文、产品名、代码、命令、字段名必须保留时，才使用相应语言；必要时可保留原文并附中文说明。

## 知识库完整记录

### 当前项目资料可见性

- 当前 `IIJ-Workspace` 项目中，用户提供的项目材料、对话结论和知识库条目默认不标记为“受限”或“客户专属”，可在后续项目中调用和复用。
- 该规则仅改变资料分类与复用边界；凭据、密码、API Key、私钥等秘密信息仍不得记录原文。

- 用户提供的项目材料、会议内容、邮件、文件、事实、参数、报价、时间节点、结论、待确认事项和后续行动，默认均应记录到知识库。
- 通用可复用内容沉淀到产品、FAQ、提案、SOP、AI 或合规目录；客户专属内容完整记录在对应客户条目或项目专属目录，并标记“客户专属”或“受限”。
- 客户专属价格、折扣、项目范围、人员规模和排期可记录在受限客户条目中，但不得复制到通用知识、跨客户材料或对外公开内容。
- 凭据、密码、API Key、私钥等秘密信息不得记录原文；应仅记录其存在、保管责任和受控存放位置。
- 来源材料包含日期或具体时间时，必须保留；对来源事实、分析推断、假设和待确认事项应明确区分。
