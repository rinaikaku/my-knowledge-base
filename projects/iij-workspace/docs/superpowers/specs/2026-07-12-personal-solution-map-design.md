# 个人 Solution Map PPT 设计

## 目标

生成一份 9 页中文个人总结 PPT，用于快速判断客户处于何种 IT/安全阶段、可匹配哪些 IIJ 解决方案，以及下一步应收集什么信息。

## 受众与边界

- 受众：作者本人，用作客户机会梳理与个人销售作战地图。
- 不包含客户名称、报价、联系人、IP、设备明细或合同条件。
- 使用现有知识库中的通用方案：现场安全调查、数据三法/等保、GIO/基础设施、SASE/网络、IP-Guard/终端、EDR/MDR/SOC、IFMS。

## 页面结构

1. 封面：IIJ China Solution Map。
2. 总体地图：调查→合规→建设→防护→运营闭环。
3. 发现与评估：中国据点安全调查、漏洞评估、资产盘点。
4. 合规与治理：数据三法、等保、数据出境、制度与证据。
5. 基础设施与云：GIO、中国服务器整合、备份恢复、运维外包。
6. 网络与访问：专线/SD-WAN、SASE、ZTNA、SWG、远程访问。
7. 终端与数据：AD/ID、IP-Guard、EDR、数据泄露防护。
8. 持续运营：SOC、MDR、IFMS、事件响应与月度治理。
9. 个人行动清单：机会识别问题、调研输入、下一步动作。

## 视觉与输出

- 16:9，IIJ 红白商务风。
- 核心视觉为生命周期环与六类解决方案卡片；不使用外部图片。
- 交付 PPTX，附带 Markdown 页面脚本。
